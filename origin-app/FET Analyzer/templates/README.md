# Origin templates

此目录预留给由 Origin 自身保存的二进制模板：

- `FET_Transfer_Publication.otpu`：论文风格 Id–Vg 图模板；
- `FET_Transfer_Analysis.ogwu`：包含图、summary 和参数设置的分析模板。

建议样式：白底、6 pt 图层边距、Arial/Helvetica 8–10 pt、主曲线 1.5 pt、拟合线 2 pt、颜色同时配合线型，避免只依赖颜色区分。左层应绘制 `logAbsId` 数据并用 `10^x` tick label 显示电流量级，不使用 Origin 的对数轴缩放。

模板需要在目标 Origin 版本中创建和验证，因此本仓库不放置伪造的空二进制模板。
